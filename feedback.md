## Day 1 — 2026-04-17 — Setup & orientation

### First impressions
- The Nox docs entry point (https://docs.iex.ec/nox-protocol/getting-started/welcome) is reachable and clearly organized around getting-started + reference.
- The confidential DeFi wizard (https://cdefi-wizard.iex.ec/) and cdefi.iex.ec demo are a strong signal: builders land on something runnable, not a wall of text.
- Discovery of the iExec NPM org (https://www.npmjs.com/org/iexec-nox) was straightforward but it would help to have one "start here" page that pins the exact packages to install for a Solidity project vs. a TypeScript project.
